Website Planning Document

1. Site Name

Suu's Papercraft Place

This name represents that the website is a place where visitors can find information and templates related to papercraft. It also incorporates the name of the business owner, represented as Suu.

Domain: suu-papercraft.pe

2. Site Purpose

To provide people with ready-made papercraft sculptures as well as printable templates they can assemble on their own.

3. Scenarios

  • What is papercraft?
  • Where can I find assembled sculptures?
  • Where can I find templates to build?

4. Color Schema

#345995 - Main blue color for the site.

#F2C14E - Gold color for accents.

5. Typography

Poppins - Used for headings (h1, h2).

Lato - Used for the general site content.
